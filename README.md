# Agentenbibliothek

**Praxisnahe KI-Workflows für Vereine, Gastronomie und Website-Pflege.**

Eine Sammlung strukturierter Arbeitsanweisungen (`.md`-Dateien) für die 
Arbeit mit KI-Assistenten wie Claude. Entwickelt von [id zemke](https://id-zemke.de) 
im Rahmen des Projekts [Agentenbibliothek](https://agentenbibliothek.de).

---

## Inhalt

| Kategorie | Beschreibung |
|---|---|
| `redaktion/` | Spielberichte, Vereinsnews, Veranstaltungen |
| `social-media/` | Posts, Teaser, Hashtags |
| `website-pflege/` | Termine, Kontakte, Hinweise |
| `gastronomie/` | Speisekarten, Angebote, Events |
| `organisation/` | Protokolle, To-dos, Rundschreiben |

## Nutzung

1. Gewünschte `.md`-Datei öffnen
2. Inhalt in Claude einfügen (oder direkt als Datei übergeben)
3. Eigene Eingaben ergänzen
4. Ergebnis erhalten und verwenden
